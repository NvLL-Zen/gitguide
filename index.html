Authenticating git with github

There are several methods to authenticate to github, one that i prefer to use is github CLI, to get started on this. Head over to the official github repo and follow the instructions to install it in your operating system.

Once installed, you should be able to type "gh" in your command line and see something similar to this:

– □ x 

~> gh

work seamlessly with GitHub from the command line.
